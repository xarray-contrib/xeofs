======================
API
======================

Methods are categorized based on their analytical focus:

1. **Single-Set Analysis**: Methods that examine relationships among variables within a single dataset, or when multiple datasets are combined and analyzed as one.
2. **Multi-Set Analysis**: Methods that investigate relationships or patterns between variables across two or more distinct datasets.
